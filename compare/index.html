สรุปครบ 3 บทเรียนในหน้าเดียว

เปรียบเทียบ 3 ระบบควบคุมสัญญาณไฟ

Fixed Time, Vehicle Actuated (VA) และ Adaptive ต่างกันอย่างไร ใช้อุปกรณ์อะไร และเหมาะกับแยกแบบไหน — ดูแบบเทียบข้างกันทีละหัวข้อ

  • Fixed Timeทำงานตามตารางเวลาที่ตั้งไว้
  • VAปรับไฟเขียวตามรถที่ตรวจจับได้จริง
  • Adaptiveปรับตามข้อมูลสด ประสานทั้งโครงข่าย

ภาพรวม

3 ระบบ 3 แนวคิด

หัวใจของแต่ละระบบในหนึ่งบรรทัด ก่อนลงรายละเอียดในตารางด้านล่าง

ระบบตั้งเวลาคงที่ (Fixed Time)

ทำงานตามตารางเวลาที่ตั้งไว้ล่วงหน้า เรียบง่าย คาดเดาได้

ระบบตรวจจับรถ (Vehicle Actuated — VA)

ใช้เซนเซอร์ตรวจจับรถ แล้วปรับไฟเขียวตามรถที่มาจริง

ระบบปรับตัวอัตโนมัติ (Adaptive)

วัดสภาพจราจรจริงต่อเนื่อง แล้วปรับแผนสัญญาณไฟและประสานหลายแยกโดยอัตโนมัติ

ตารางเปรียบเทียบ

เทียบข้างกัน ทีละหัวข้อ

เปรียบเทียบทั้ง 3 ระบบในหัวข้อเดียวกัน ตั้งแต่หลักการทำงาน อุปกรณ์ที่ต้องใช้ ไปจนถึงตัวอย่างการใช้งานจริง

ตารางเปรียบเทียบระบบควบคุมสัญญาณไฟจราจร 3 ระบบ — Fixed Time, Vehicle Actuated (VA) และ Adaptive — ในหัวข้อหลักการทำงาน อุปกรณ์ที่ต้องใช้ ความเหมาะสม จุดแข็ง ข้อจำกัด และตัวอย่างการใช้งาน
ประเภทระบบ (System Type)หลักการทำงาน (How It Works)อุปกรณ์ที่ต้องใช้ (Equipment)เหมาะกับ (Best For)จุดแข็ง (Strength)ข้อจำกัด (Limitation)ตัวอย่างการใช้งาน (Example)
ระบบตั้งเวลาคงที่ (Fixed Time)ทำงานตามตารางเวลาที่ตั้งไว้ล่วงหน้า ซ้ำเดิมทุกรอบตู้ควบคุมสัญญาณไฟพื้นฐาน ไม่ต้องใช้อุปกรณ์ตรวจจับ (Detector)แยกที่ปริมาณรถคงที่และมีรูปแบบจราจรซ้ำทุกวันเรียบง่าย ต้นทุนต่ำ คาดเดาได้ และประสานสัญญาณระหว่างแยกได้ง่ายไม่ตอบสนองต่อสภาพจราจรจริง ทำให้รอคอยโดยไม่จำเป็นเมื่อรถน้อยแยกในเขตเมืองที่มีรูปแบบจราจรซ้ำทุกวัน เช่น ย่านสำนักงานช่วงเร่งด่วน
ระบบตรวจจับรถ (Vehicle Actuated — VA)ใช้ Detector ตรวจจับรถ แล้วให้หรือต่อขยายไฟเขียวตามรถที่มาจริง ภายใต้กรอบเวลาขั้นต่ำ–สูงสุดตู้ควบคุม + อุปกรณ์ตรวจจับรถ เช่น ขดลวดฝังถนน เรดาร์ หรือกล้องแยกที่จราจรผันผวน โดยเฉพาะแยกถนนหลักตัดถนนรองที่รถไม่เท่ากันลดการรอคอยที่ไม่จำเป็น ข้ามเฟสที่ไม่มีรถ ยืดหยุ่นตามจราจรแต่ละรอบพึ่งพา Detector สูง มองเห็นเฉพาะแยกตัวเอง และเมื่อรถแน่นทุกทิศทางจะคล้าย Fixed Timeแยกถนนหลักตัดถนนรองชานเมือง ที่ถนนรองมีรถมาเป็นช่วง ๆ
ระบบปรับตัวอัตโนมัติ (Adaptive)วัดสภาพจราจรจริงต่อเนื่อง แล้วปรับรอบสัญญาณ เวลาเขียว และการประสานหลายแยกโดยอัตโนมัติตู้ควบคุม + Detector หลายจุด + ซอฟต์แวร์ประมวลผล + ระบบสื่อสารเชื่อมหลายแยกเส้นทางหลักและโครงข่ายเมืองที่จราจรผันผวนสูง ต้องการประสานหลายแยกตอบสนองสถานการณ์จริงได้ดีที่สุด และประสานทั้งเส้นทางหรือโครงข่ายได้ต้นทุนและความซับซ้อนสูงสุด พึ่งพาคุณภาพ Detector และไม่ได้ทำให้รถหายติดถ้าปริมาณรถเกินความจุถนนถนนสายหลักในเมืองหลายแยกต่อเนื่อง ที่ปริมาณรถเปลี่ยนตามช่วงเวลาและเหตุการณ์

เคล็ดลับ: เลื่อนตารางไปทางขวาเพื่อดูครบทุกหัวข้อ — คอลัมน์ชื่อระบบจะติดอยู่ด้านซ้ายเสมอ

ระบบตั้งเวลาคงที่ (Fixed Time)

หลักการทำงาน (How It Works)
ทำงานตามตารางเวลาที่ตั้งไว้ล่วงหน้า ซ้ำเดิมทุกรอบ
อุปกรณ์ที่ต้องใช้ (Equipment)
ตู้ควบคุมสัญญาณไฟพื้นฐาน ไม่ต้องใช้อุปกรณ์ตรวจจับ (Detector)
เหมาะกับ (Best For)
แยกที่ปริมาณรถคงที่และมีรูปแบบจราจรซ้ำทุกวัน
จุดแข็ง (Strength)
เรียบง่าย ต้นทุนต่ำ คาดเดาได้ และประสานสัญญาณระหว่างแยกได้ง่าย
ข้อจำกัด (Limitation)
ไม่ตอบสนองต่อสภาพจราจรจริง ทำให้รอคอยโดยไม่จำเป็นเมื่อรถน้อย
ตัวอย่างการใช้งาน (Example)
แยกในเขตเมืองที่มีรูปแบบจราจรซ้ำทุกวัน เช่น ย่านสำนักงานช่วงเร่งด่วน

ระบบตรวจจับรถ (Vehicle Actuated — VA)

หลักการทำงาน (How It Works)
ใช้ Detector ตรวจจับรถ แล้วให้หรือต่อขยายไฟเขียวตามรถที่มาจริง ภายใต้กรอบเวลาขั้นต่ำ–สูงสุด
อุปกรณ์ที่ต้องใช้ (Equipment)
ตู้ควบคุม + อุปกรณ์ตรวจจับรถ เช่น ขดลวดฝังถนน เรดาร์ หรือกล้อง
เหมาะกับ (Best For)
แยกที่จราจรผันผวน โดยเฉพาะแยกถนนหลักตัดถนนรองที่รถไม่เท่ากัน
จุดแข็ง (Strength)
ลดการรอคอยที่ไม่จำเป็น ข้ามเฟสที่ไม่มีรถ ยืดหยุ่นตามจราจรแต่ละรอบ
ข้อจำกัด (Limitation)
พึ่งพา Detector สูง มองเห็นเฉพาะแยกตัวเอง และเมื่อรถแน่นทุกทิศทางจะคล้าย Fixed Time
ตัวอย่างการใช้งาน (Example)
แยกถนนหลักตัดถนนรองชานเมือง ที่ถนนรองมีรถมาเป็นช่วง ๆ

ระบบปรับตัวอัตโนมัติ (Adaptive)

หลักการทำงาน (How It Works)
วัดสภาพจราจรจริงต่อเนื่อง แล้วปรับรอบสัญญาณ เวลาเขียว และการประสานหลายแยกโดยอัตโนมัติ
อุปกรณ์ที่ต้องใช้ (Equipment)
ตู้ควบคุม + Detector หลายจุด + ซอฟต์แวร์ประมวลผล + ระบบสื่อสารเชื่อมหลายแยก
เหมาะกับ (Best For)
เส้นทางหลักและโครงข่ายเมืองที่จราจรผันผวนสูง ต้องการประสานหลายแยก
จุดแข็ง (Strength)
ตอบสนองสถานการณ์จริงได้ดีที่สุด และประสานทั้งเส้นทางหรือโครงข่ายได้
ข้อจำกัด (Limitation)
ต้นทุนและความซับซ้อนสูงสุด พึ่งพาคุณภาพ Detector และไม่ได้ทำให้รถหายติดถ้าปริมาณรถเกินความจุถนน
ตัวอย่างการใช้งาน (Example)
ถนนสายหลักในเมืองหลายแยกต่อเนื่อง ที่ปริมาณรถเปลี่ยนตามช่วงเวลาและเหตุการณ์

แนวทางการเลือก

เลือกระบบไหนดี?

ไม่มีระบบใด “ดีที่สุด” สำหรับทุกแยก เริ่มจากคำถามง่าย ๆ ว่าจราจรผันผวนแค่ไหน แล้วตามเส้นทางที่ตรงกับสถานการณ์ของคุณ
